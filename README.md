# KM Direct

Web app proprietaria di KM Kebab Mediterraneo per ordini diretti (delivery e
ritiro), sviluppata a partire dalla KM Direct Master Specification.

## Stato

Fase iniziale: scaffold Next.js minimo, pubblicato su Vercel per verificare
il collegamento GitHub → Vercel. Il menu, il carrello e il checkout vengono
aggiunti nei passaggi successivi.

## File di riferimento del progetto

- `km_direct_schema.sql` — schema database (Supabase/PostgreSQL)
- `km_direct_design_system.md` — palette colori, tipografia, regole componenti

## Sviluppo locale (facoltativo, richiede Node.js)

```
npm install
npm run dev
```
